# 敌星弹雨 Infinity Battle
## 简介
使用 Java 实现的简单 2D 游戏引擎。


## 主要模块
* **ib-api**  核心接口
* **ib-core**  核心实现
* **ib-container**	  容器实现
* **ib-character**  游戏角色实现
* **ib-component**  游戏组件实现
* **ib-start**  游戏入口
